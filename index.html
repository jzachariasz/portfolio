Project nr. 1: MindCircle

MindCircle is a web application where users can browse, create and review gatherings. Key features include: user authentication, gathering listings, add & edit gathering, reviews system, cloudinary image upload, data validation & security, profile editing.

Project nr. 2: YelpCamp

YelpCamp is a full-stack web application I built as a part of the Colt Steele's Web Developer Bootcamp course. It is my first application of this kind I've ever built. I followed along with the instructor while also solving bugs and implementing fixes of my own, which helped deepen my understanding of the stack. The app allows user to create, view, edit and review campgrounds. It includes features like: geolocation and coordinates, authentication and authorization, campground management, review system, form validation, image upload and deletion via "Cloudinary" and security features like helmet, mongo sanitize and input validation. This project gave me hands-on experience with technologies like Express, MongoDB, Mongoose, Passport and EJS.

Stack used: Bootstrap, Node.js, MongoDB, Mongoose, Express, Passport.js, Cloudinary, Helmet
The application needs a second to load https://yelpcamp-3hps.onrender.com/
